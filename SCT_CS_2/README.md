# 🖼 Image Encryption Tool

## 📌 Description
This project is a simple image encryption and decryption tool developed using Python.

The program encrypts images using pixel manipulation techniques and restores them back during decryption.

---

## 🛠 Features
- Encrypt images
- Decrypt encrypted images
- Pixel-level manipulation
- XOR-based encryption technique

---

## ▶️ How to Run

```bash
python image_encryptor.py
```

---

## 📷 Output
The folder contains:
- Original image
- Encrypted image
- Decrypted image

## 📚 Concepts Used
- Image Processing
- Pixel Manipulation
- XOR Encryption
- Python Pillow Library
