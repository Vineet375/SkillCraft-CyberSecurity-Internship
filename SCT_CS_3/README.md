# 🔑 Password Strength Checker

## 📌 Description
This project is a Python-based password strength checker developed during the Cyber Security Internship at SkillCraft Technology.

The tool analyzes password strength based on various security criteria.

---

## 🛠 Features
- Checks password length
- Detects uppercase letters
- Detects lowercase letters
- Detects numbers
- Detects special characters
- Displays password strength level

---

## ▶️ How to Run

```bash
python password_checker.py
```

---

## 📷 Output
The screenshots folder contains output examples of weak, medium, and strong passwords.
