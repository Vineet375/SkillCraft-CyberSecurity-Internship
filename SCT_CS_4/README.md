# ⌨️ Basic Keylogger

## 📌 Description
This project is a basic educational keylogger developed using Python as part of the Cyber Security Internship at SkillCraft Technology.

The program records keystrokes and stores them locally in a text file.

⚠️ This project is developed strictly for educational and ethical cybersecurity learning purposes only.

---

## 🛠 Features
- Captures keyboard keystrokes
- Saves logs into a local file
- Lightweight and simple implementation

---

## ▶️ How to Run

```bash
python keylogger.py
```
