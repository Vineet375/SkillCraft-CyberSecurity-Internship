# 🔐 Caesar Cipher Encryption & Decryption

## 📌 Description
This project is a Python-based Caesar Cipher program developed as part of the Cyber Security Internship at SkillCraft Technology.

The program allows users to encrypt and decrypt messages using a custom shift value.

---

## 🛠 Features
- Encrypt text messages
- Decrypt encrypted messages
- Custom shift value support
- User-friendly command-line interface

---

## ▶️ How to Run

```bash
python caesar_cipher.py
```
